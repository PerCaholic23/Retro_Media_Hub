# Retromedia Hub

## ข้อควรรู้
version นี้ปรับโครงสร้างใหม่ทั้งหมด โดยการแยก backend และ frontend ออกจากกัน เนื่องจาก version ก่อนหน้าทั้งสองอันปนกันใน directory เดียว ละก็ backend กับ frontend รันแยกกัน

## การทำงานในแต่ละขั้นตอน
### ขั้นตอนแรก
โคลน Project มาลงก่อนนะจ๊ะ

### Backend
1) cd backend
2) npm install
3) node server.js

### Frontend
1) cd frontend
2) npm install
3) npm start
